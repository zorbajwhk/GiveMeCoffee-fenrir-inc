# 簡易仕様書サンプル

### 作者
Zorba Jobs Wong
### アプリ名
GiveMeCoffee

#### コンセプト
コーヒーラヴァーが日本だけではなく、
全球どこでものコーヒー飲める場所情報をどんな設備でも、高速で、シンプルで、フレキシブルで検索できるようになる・


#### こだわったポイント
・ プログレッシブウェブアプリ (PWA)：インストレーションしなくでも、ネイティブアプリのような見た目で使えるデザイン。
・ アダプティブウェブデザイン (AWD)：ユーザーの最高な体験のためにパソコン、タブレット、スマホ、全て完全対応している。
・ React.js コードベース：業界マーケットシェア最高なフレームワークで作った。
・ サーバーレス：AWS Amplifyと Cloudflare Workerを利用して、最高水準のサービスかど最低水準の運用コストを目指します。
・ フレキシブル：ファンククリックデザイン。

## アプリケーション機能

### 機能一覧
- おすすめ順検索：Yelp APIを使用して、入力した辺の周辺のコーヒーを飲める場所をおすすめ順で検索する。
- 評価順検索：Yelp APIを使用して、入力した辺の周辺のコーヒーを飲める場所を評価順で検索する。
- 口コミ順検索：Yelp APIを使用して、入力した辺の周辺のコーヒーを飲める場所を口コミ順で検索する。
- 地図アプリ連携：店の住所を押すと、Google Mapを開けます。
- 口コミサイト連携：店の口コミ数を押すと、店のYelpページを開けます。
- プログレッシブウェブアプリ (PWA)：永遠に最新版のアプリをユーザーに提供できるために。
- アダプティブウェブデザイン (AWD)：最高な体験のためにパソコン、タブレット、スマホ、全て完全対応している。

### 画面一覧
- 検索&一覧画面 ：場所を入力して、レッツゴーボタンを押して、検索結果のコーヒーを飲める場所の一覧表示する。

### 使用しているAPI,SDK,ライブラリなど
- Yelp API: 店の情報をFetchする。
- Google Cloud Translate API - 入力した場所は英語、または日本語も対応できるようになる。
